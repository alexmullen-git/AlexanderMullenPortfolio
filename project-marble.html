Marble Track System

Mounting and track hardware for customizable marble runs on a pegboard wall

Overview

This project is a universal marble track system for a standard pegboard. 3D printed mounts secure wooden dowels to the pegboard, pivot mounts attach to the dowels, track mounts with dovetail joints connect to the pivot mounts, and the track clips into the mounts allowing the user to shape their own track then use the mounts to secure it. The focus was printing simplicity, low-profile hardware, and ease of use. See the interactive 3D models and demo tracks below!
